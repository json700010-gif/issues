# LineageOS Issue Tracker
This repository is for **bug reports** only and uses a structured issue form with automated validation.

## How to report a bug
1. Check existing issues for duplicates.
2. Read: https://wiki.lineageos.org/how-to/bugreport
3. Open a new issue using the **Bug Report** template.
4. Fill all required fields accurately:
   - Device codename
   - Exact LineageOS version
   - Build date (`YYYYMMDD`)
   - Kernel version
   - Expected behavior
   - Current behavior
   - Steps to reproduce

## What not to report here
- Unofficial build bugs
- Missing builds
- Website problems
- Device support requests
- Feature requests

For general support: https://lineageos.org/community

## Automated checks
On open/reopen, a workflow validates your report format and values (device/version/date), applies labels, and assigns maintainers when possible.

If your report is invalid, it will be closed with bot feedback. Fix the issue details and comment:

`/reopen`

to run validation again.
